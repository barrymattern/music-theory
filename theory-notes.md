# Music Theory Notes


## Circle 4ths/5ths
* Circle of 4ths = counter-clockwise
    * G -> C = 4th
* Circle of 5ths = clockwise
    * C -> G = 5th
* [Diagram][1]
    * Includes harmonic minors


## Key Signatures
* Great tip from [Victor Wooten][2]
    * Keys with same name should add up to 7
        * AMaj  = 3 sharps
        * AbMaj = 4 flats
